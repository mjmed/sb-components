# mj-my-storybook-components


Este paquete es útil para practicar **React Components** + **Storybook** + **NPM Publish**.


## Instalación

Usando npm:  
```
npm install mj-my-storybook-components
```

Usando yarn:  
```
yarn add mj-my-storybook-components
```
